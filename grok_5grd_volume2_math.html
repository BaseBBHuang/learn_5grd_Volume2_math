公式速查表 · 核心10大概念

仅限五年级下册，最重要10个概念/公式，每条附一句话解释 + 简单例子。随时查！

概念 / 公式 一句话解释 简单例子
因数和倍数 整数除法中商是整数且余数为0，除数就是被除数的因数，被除数就是除数的倍数 12÷2=6 → 2是12的因数，12是2的倍数
2的倍数 个位上是0、2、4、6、8的数都是2的倍数（偶数） 24、36、100（个位都是偶数）
3的倍数 各位数字之和是3的倍数的数就是3的倍数 2+4=6（是3的倍数）→ 24是3的倍数
质数与合数 只有1和它本身两个因数的数是质数；除了1和本身还有其他因数的是合数 2、3、5是质数；4、6、9是合数
长方体表面积 6个面的面积之和 = 2×(长×宽 + 长×高 + 宽×高) 长6、宽5、高4 → 2×(30+24+20)=148
正方体表面积 6个面完全相同 = 6×棱长² 棱长5 → 6×25=150
长方体体积 长×宽×高（物体占空间的大小） 长6、宽4、高5 → 120
正方体体积 棱长³（特殊长方体体积） 棱长4 → 64
分数意义 把单位“1”平均分成若干份，表示这样的一份或几份的数 把1平均分成5份，取3份 → 3/5
分数加减法 同分母直接分子相加减；异分母先通分再加减 1/3 + 1/6 = 2/6 + 1/6 = 3/6 = 1/2

数学原来这么有用 · 生活场景小手册

每个知识点配一个日常真实例子 + 小学生能看懂的解释，最后一句话总结“所以学这个有什么用”

🛒

因数和倍数

妈妈买了24个苹果，要平均分给班上6个小朋友，每人分几个？24是6的倍数，6是24的因数！

所以学这个有什么用：分东西、排队、买礼物时不会算错份数啦！

📦

长方体体积

你的书包长40cm、宽25cm、高15cm，能装多少立方厘米的东西？用长×宽×高就知道啦！

所以学这个有什么用：买新书包、装玩具、搬家时知道空间够不够！

🍰

分数的意义

一个蛋糕平均切成8块，你吃了3块，就是3/8。把“1”平均分，拿几份就是几分之几！

所以学这个有什么用：分零食、分披萨、算考试成绩占比，再也不怕啦！

📈

折线统计图

你记录一周每天做作业的时间，用折线图一看，哪天最累就清楚了，还能预测下周！

所以学这个有什么用：管理时间、看天气、了解成绩变化，生活更聪明！

🏠

长方体表面积

给礼物盒包纸，需要包6个面。长20cm、宽15cm、高10cm，用公式算出总面积就够啦！

所以学这个有什么用：包装礼物、做手工、刷墙时知道要多少材料！

🔍

找次品（数学广角）

一盒12个球，有一个是次品，用天平称3次就能找出！学会推理，像侦探一样聪明。

所以学这个有什么用：玩游戏、解决问题、考试时逻辑更强！

🛡️

避坑指南 · 最常见的7大易错点

像好朋友一样提醒你：这些地方最容易丢分！每个坑都有错误示范 + 正确做法 + 记忆小口诀

1️⃣
忘记1和本身也是因数
❌ 错误示范：

18的因数只有2、3、6、9

✅ 正确做法：

18的因数：1、2、3、6、9、18

🔥 记忆口诀：因数找全，1和它本身千万别漏！

2️⃣
3的倍数只看个位
❌ 错误示范：

24不是3的倍数（个位4）

✅ 正确做法：

2+4=6（是3的倍数）

🔥 记忆口诀：3的倍数看各位数字和，不是看个位！

3️⃣
表面积少算相对面
❌ 错误示范：

长方体只算3个面面积

✅ 正确做法：

2×(长×宽+长×高+宽×高)

🔥 记忆口诀：表面积要乘2，相对面一对一对算！

4️⃣
分数加减不先通分
❌ 错误示范：

1/2 + 1/3 = 2/5（错）

✅ 正确做法：

3/6 + 2/6 = 5/6

🔥 记忆口诀：异分母，先通分，分子相加减！

5️⃣
体积和表面积混淆
❌ 错误示范：

正方体体积=6×棱长²（错）

✅ 正确做法：

体积=棱长³，表面积=6×棱长²

🔥 记忆口诀：体积“³”三维空间，表面积“²”六个面！

6️⃣
奇偶数运算记反
❌ 错误示范：

奇数+奇数=偶数（错）

✅ 正确做法：

奇+奇=偶，奇+偶=奇

🔥 记忆口诀：奇奇得偶，奇偶得奇，偶偶还是偶！

7️⃣
找次品不按最优策略
❌ 错误示范：

每次只称一个球

✅ 正确做法：

分组称（3 vs 3），3次找出次品

🔥 记忆口诀：分组称天平，信息最多最聪明！
